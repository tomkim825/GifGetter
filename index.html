G-Rated Gif Getter [keep it SFW]
Hello there!
This project uses AJAX to call the Giphy API. While the initial buttons are breakfast related, feel free to add any buttons through the input feature above. Click on any button to display gifs. Gifs start non-animated. Click to show the animated version. Page refreshes will change the header color theme. Each gif button press will also make a random color theme box. Thanks for looking at my site.
